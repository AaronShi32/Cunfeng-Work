## 京能数产 vs 北京国际算力服务（京算）

| 维度 | 京能数产 | 北京国际算力服务（京算） |
|------|---------|----------------------|
| **成立时间** | 2023.07 | 2023.10 |
| **注册资本** | 62.3 亿 | 1 亿（实缴 5000 万） |
| **母公司** | 北京能源集团（市国资委直管） | 北京亦庄智能城市研究院集团（亦庄控股体系） |
| **定位** | 京能集团数字化转型 + 算力新增长极 | 北京市算力资源调度服务运营主体 |
| **核心业务** | 智算基建 + 数字能源 + 企业数字化软件 | 智算中心运营 + AI 数据服务 + 大模型训推 |
| **算力规模** | 数百~千P 级（上庄公共算力平台） | 5000P（亦庄智算集群，已建成） |
| **标杆项目** | 北京人工智能公共算力平台（上庄） | 亦庄公共智算中心 + 全国首个 AI 数据训练基地 |
| **产品线** | "京能星曜"系列（天枢/开阳/天玑/玉衡/瑶光） | DeepSeek API 服务 + "模数世界"孵化器 |
| **差异化** | **电算融合**——依托京能电力资源做绿电算力 | **数据先行区**——配合国家数据基础制度试点 |
| **客户** | 200+ 家（企业/科研/政府） | 百度、智谱等头部 AI 企业 |
| **团队规模** | 50~100 人，互联网/科技背景为主 | 规模较小，偏运营 |
| **优势** | 母公司资本雄厚（62 亿）+ 能源成本优势 | 政策红利（数据先行区）+ 亦庄产业生态 |

### 核心差异总结

- **京能数产**：能源集团做算力，核心优势是**电力成本和绿电供给**，走"电算融合"路线，资本实力更强（62 亿 vs 1 亿），产品线更全（从基建到软件到行业模型）
- **京算公司**：经开区做算力，核心优势是**政策先行（数据制度试点）+ 产业集聚（亦庄 AI 企业密集）**，已建成更大规模集群（5000P），更偏公共服务平台定位

> 两家都是 2023 年成立的北京国资算力平台，但一个从**能源侧**切入（有电就有算力成本优势），一个从**产业园区侧**切入（有场景和客户密度优势）。

## 北京数据集团 vs 北京能源集团（京能）

| 维度 | 北京数据集团 | 北京能源集团（京能） |
|------|------------|-------------------|
| **成立时间** | 2025.07（揭牌 2025.09） | 1993 年前身，2004 年合并组建 |
| **注册资本** | 60 亿 | — |
| **总资产** | ~136 亿 | **4934 亿**（2024） |
| **年营收** | 初创期，未披露 | **975 亿**（2024） |
| **员工** | 初创整合阶段 | 3.4 万人 |
| **控股方** | 北京国资公司（100%） | 北京市政府直接出资（国有独资） |
| **行业** | 数据要素 / 数字经济 | 能源（电力+热力+煤炭+清洁能源） |
| **定位** | 数据全产业链平台——数据基建、交易、流通、治理 | 首都能源保障主力军 + 综合能源服务集团 |
| **上市公司** | 无 | 6 家（京能电力、京能清洁、昊华能源等） |
| **核心资产** | 首都信息、数字认证、北数所、亦庄智能院、北方算网 | 5490 万千瓦装机、6 亿㎡供热面积 |
| **与算力的关系** | 整合亦庄智能院（京算公司母公司）→ 算力是核心业务之一 | 子公司京能数产做智算 → 算力是新增长极 |
| **算力切入点** | 数据交易 + 数据基建 + 公共算力平台运营 | **电算融合**——自有电力做绿电算力降本 |
| **战略方向** | "1+4+1"：政企数字化 + 数据基建/开发/交易/生态 | "3+2+N"：电力热力煤炭 + 康养文旅 + 智算 |
| **500 强排名** | 新成立，暂无 | 中国企业 500 强第 247 位 |
| **优势** | 政策红利（数据要素改革主体）+ 数据牌照资源 | 资本雄厚 + 能源成本优势 + 成熟运营体系 |

### 核心差异

- **北京能源集团**：老牌巨无霸（4934 亿资产、30 年历史），主业是电力和供热，做算力是"用自己的电发自己的算力"——**成本侧优势**
- **北京数据集团**：2025 年新组建的数据要素国家队（136 亿资产），整合了数据交易、政务数据、算力平台等——**牌照和数据侧优势**

> **一句话**：京能是"有电的人做算力"，数据集团是"有数据的人做算力"。两者在北京智算生态里互补——京能提供绿电和算力基建，数据集团提供数据流通和算力调度运营。

### 关联关系

```
北京市国资委
├── 北京能源集团（京能）──→ 京能数产（智算基建，上庄算力平台）
│                              电算融合，绿电优势
└── 北京国资公司 ──→ 北京数据集团 ──→ 亦庄智能院 ──→ 京算公司（5000P 智算中心）
                      数据交易+治理+公共算力运营
```
